Website

What is a website?

A website is a collection of web pages and related content that is identified by a common domain name and published on at least one web server

In a simple website, there are two key components: the client and the server. The client sends a request (such as by visiting "domain.com") to the server, and the server responds by delivering the relevant files.

Key Concepts

  • Domain: A unique name for a website on the internet.
  • SEO (Search Engine Optimization): The process of optimizing a website to improve its ranking in search engine results, such as on Google.

Websites can be classified into two types

  1. Static Website: Involves only the client and the server, where content does not change based on user interaction.
  2. Dynamic Website: Involves a database that allows the website to update its content based on user interactions or other factors. The server retrieves the updated data from the database and delivers it to the client.

HTML Basic

In HTML, the structure of the content is defined by tags, attributes, and values.

  • Tags: Enclosed in angle brackets, tags define the structure of the content. For example, < html > defines the beginning of an HTML document.
  • Attributes: Provide additional information about an element. For example, the src attribute in an < img > tag specifies the image source.
  • Values: Used to specify the content of an element. For example, the value of the alt attribute in an < img > tag provides alternative text for an image.

Usefull tags

  • < h1 > - < h6 >: Headings of different sizes.
  • < p >: Paragraphs of text.
  • < a >: Links to other web pages or resources.
  • < img >: Images.
  • < ul >, < ol >, < li >: Unordered lists, ordered lists, and list items.
  • < table >, < tr >, < td >: Tables, table rows, and table data cells.
  • < span > Used to separate parts of a text, for example, making part of a paragraph bold. With a span tag, you can apply styling (like bold) to just the middle portion of a sentence
  • < div > Used to group elements together in a block.
